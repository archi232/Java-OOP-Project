# Java OOP Projects
This repository contains two Java projects developed using Object-Oriented Programming (OOP) concepts.

## Project 1: Library Management System
- Manages library book records and member details.
- Supports book issue and return operations.
- Demonstrates OOP concepts such as classes, objects, encapsulation, inheritance, and polymorphism.

## Project 2: Employee Payroll Management System
- Manages employee information and payroll records.
- Calculates employee salaries based on payroll data.
- Implements core Java and OOP principles for efficient data management.

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
